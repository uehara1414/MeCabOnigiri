# Overview

mecab-python3 wrapper

# Setup

## Requirements

- Python 3.6+

## Installation

Install MeCabOnigiri with pip:

```sh
$ pip install MeCabOnigiri
```

# Usage

After installation, the package can imported:

```sh
$ python
>>> import MeCabOnigiri
```

## Test
```sh
pipenv install --dev
pipenv run nosetests
```
